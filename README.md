## 📦 What is embr.el?

embr.el is a desktop app for Windows that uses two parts to show your content:

- Emacs acts as the display server
- Headless Chromium handles the rendering

In simple terms, embr.el lets Emacs manage the app window while Chromium does the page display work in the background. This setup can help if you want a lighter browser-style app flow with Emacs handling the frame.
